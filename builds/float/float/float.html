Unity Web Player | Float

You start attached to a handle, click other handles (highlighted in yellow) to jump toward them.

White objects need to be pushed to handles so they can be attached.

You win if all objects are attached to handles.

If attached to a handle, you can only head towards another handle.

If you are in flight and near a wall, you can push off it in the direction you click (anywhere in the room).

It is possible to get stranded :(

It may be impossible (or very difficult) to finish some rooms :(

Win detection is not implemented :(
